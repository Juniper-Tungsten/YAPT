# About #
YAPT (Yet Another Provisioning Tool) is for provisioning

- SRX / vSRX
- MX / vMX
- NFX
- EX
- QFX

devices. It's main goal is to deliver Zero Touch Provisioning.

YAPT runs with **Python 2 (2.7+)**.
